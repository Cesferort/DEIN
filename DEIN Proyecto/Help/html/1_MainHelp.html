Proyecto de DEIN

Este programa permitirá al usuario gestionar una base de datos de Olimpiadas. El acceso se restringirá a usuarios creados por el administrador, de esta forma, en busca de mucha mayor seguridad, podremos guardar todos los cambios producidos por cada empleado. Tras iniciar sesión, haciendo uso de un menú, el usuario podrá acceder a las diferentes secciones para ejercer su trabajo de forma simplificada. Este menú se repartirá en tres secciones, que serían:

  • File
  • Olimpiada
  • Opciones
